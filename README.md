## 🚀 Website deployen

1. Geh auf [netlify.com](https://netlify.com) → Kostenlosen Account erstellen
2. Zieh den `website/` Ordner per Drag & Drop aufs Netlify Dashboard
3. Fertig – deine Seite ist live unter `xyz.netlify.app`
4. Optional: Eigene Domain verbinden (z.B. `pmnhg.de`)

## 📊 GuV-Rechnung

Die Excel-Datei `docs/guv-rechnung-2026.xlsx` enthält 3 Sheets:

| Sheet | Inhalt |
|-------|--------|
| **GuV Monatsübersicht** | Einnahmen & Ausgaben pro Monat, Gewinn/Verlust, Marge, Gewinn pro Person |
| **Kundenliste** | Tracker für alle Kunden mit Paket, Betrag, Bezahlt-Status |
| **Equipment & Kosten** | Übersicht eures gesamten Equipments |

**Blaue Zahlen** = Eingabefelder → hier tragt ihr eure echten Werte ein  
**Schwarze Zahlen** = Formeln → berechnen sich automatisch
